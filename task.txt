## Running Locally

### Backend (currently running on port 8000)
```bash
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### Frontend
```bash
cd frontend
npm run dev
```

## API Endpoints
- GET /health - Health check
- GET /search?q=keyword&limit=50 - Search reviews
- POST /analyze - Analyze sentiment

## Notes
- ML Accuracy: 89.11%
- Chart library: Recharts
- Dataset: 568,454 Amazon food reviews (200k sample used for training)
